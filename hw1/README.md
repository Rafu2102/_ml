# 旅行推銷員問題（TSP）- 爬山演算法實作

本專案是一個使用 Python 撰寫的簡單程式，旨在透過 **爬山演算法（Hill Climbing Algorithm）** 來解決經典的 **旅行推銷員問題（TSP）**。

程式 [Climbing TSP.py](https://github.com/Rafu2102/_ml/blob/master/hw1/Climbing%20TSP.py) 會隨機產生 `8` 個地點的座標，並嘗試尋找拜訪所有地點且最後回到起點的最短路徑。

---

## 演算法原理

本程式運作的**爬山演算法**核心邏輯如下：

1. **初始狀態**：隨機產生一條拜訪所有城市的起點路線。
2. **尋找鄰居**：透過隨機交換兩個城市的位置，產生稍微不同的微調路線。
3. **評估移動**：計算新路線的總距離。如果距離比目前路線更短，就移動並換成新路線。
4. **終止條件**：重複上述微調與移動步驟，直到無論怎麼交換城市都無法讓距離更短為止（代表到達山頂，已找到最佳解）。

---

## 如何執行

請在終端機中執行以下指令以啟動程式：

```powershell
python "Climbing TSP.py"
```
